# ACR TI-RADS — Scoring Algorithm Reference

## Citation
Tessler FN, Middleton WD, Grant EG, et al. ACR Thyroid Imaging, Reporting and Data System (TI-RADS): White Paper of the ACR TI-RADS Committee. Journal of the American College of Radiology. 2017;14(5):587-595. DOI: 10.1016/j.jacr.2017.01.046. PMID: 28372962.

## Scoring Categories (choose ONE per category, except Echogenic Foci)

### Composition
| Feature | Points |
|---------|--------|
| Cystic or almost completely cystic | 0 |
| Spongiform | 0 |
| Mixed cystic and solid | 1 |
| Solid or almost completely solid | 2 |

### Echogenicity
| Feature | Points |
|---------|--------|
| Anechoic | 0 |
| Hyperechoic or isoechoic | 1 |
| Hypoechoic | 2 |
| Very hypoechoic | 3 |

### Shape
| Feature | Points |
|---------|--------|
| Wider-than-tall | 0 |
| Taller-than-wide | 3 |

### Margin
| Feature | Points |
|---------|--------|
| Smooth | 0 |
| Ill-defined | 0 |
| Lobulated or irregular | 2 |
| Extra-thyroidal extension | 3 |

### Echogenic Foci (choose ALL that apply — ADDITIVE)
| Feature | Points |
|---------|--------|
| None or large comet-tail artifacts | 0 |
| Macrocalcifications | 1 |
| Peripheral (rim) calcifications | 2 |
| Punctate echogenic foci | 3 |

## Point Thresholds
| Points | Level | Designation |
|--------|-------|-------------|
| 0 | TR1 | Benign |
| 2 | TR2 | Not Suspicious |
| 3 | TR3 | Mildly Suspicious |
| 4-6 | TR4 | Moderately Suspicious |
| >= 7 | TR5 | Highly Suspicious |

Note: 1 point maps to TR2 as fallback (practically impossible).

## Management
| Level | FNA if >= | Follow-up if >= | Below threshold |
|-------|-----------|-----------------|-----------------|
| TR1 | No FNA | No follow-up | — |
| TR2 | No FNA | No follow-up | — |
| TR3 | 2.5 cm | 1.5 cm | < 1.5 cm: none |
| TR4 | 1.5 cm | 1.0 cm | < 1.0 cm: none |
| TR5 | 1.0 cm | 0.5 cm | < 0.5 cm: none |
