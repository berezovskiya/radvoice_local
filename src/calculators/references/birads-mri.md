# ACR BI-RADS MRI — Classification Reference

## Citation
ACR BI-RADS Atlas: Breast Imaging Reporting and Data System, 5th Edition. American College of Radiology, Reston, VA, 2013.

## Assessment Categories
| Category | Name | Malignancy % | Management |
|----------|------|-------------|------------|
| 0 | Incomplete | N/A | Additional imaging needed |
| 1 | Negative | ~0% | Routine screening |
| 2 | Benign | ~0% | Routine screening |
| 3 | Probably Benign | <= 2% | Short-interval follow-up (6 months) |
| 4A | Low Suspicion | > 2-10% | Biopsy recommended |
| 4B | Moderate Suspicion | > 10-50% | Biopsy recommended |
| 4C | High Suspicion | > 50-< 95% | Biopsy recommended |
| 5 | Highly Suggestive | >= 95% | Biopsy strongly recommended |
| 6 | Known Malignancy | 100% | Treatment monitoring |

## MRI Mass Descriptors

### Shape
- Oval (benign)
- Round (benign)
- Irregular (suspicious)

### Margin
- Circumscribed (benign)
- Not circumscribed — Irregular (suspicious)
- Not circumscribed — Spiculated (suspicious)

### Internal Enhancement
- Homogeneous (benign)
- Heterogeneous (suspicious)
- Rim enhancement (suspicious)
- Dark internal septations (benign)

## Non-Mass Enhancement (NME) Descriptors

### Distribution
- Focal (lower suspicion)
- Linear (intermediate)
- Segmental (higher suspicion)
- Regional (intermediate)
- Multiple regions (intermediate)
- Diffuse (higher suspicion)

### Internal Enhancement Pattern
- Homogeneous (lower suspicion)
- Heterogeneous (intermediate)
- Clumped (higher suspicion)
- Clustered ring (higher suspicion)

## Kinetic Curve Assessment

### Initial Phase
- Slow (benign)
- Medium (indeterminate)
- Fast (suspicious)

### Delayed Phase
- Persistent / Type I (benign)
- Plateau / Type II (indeterminate)
- Washout / Type III (suspicious)

## Scoring Logic for Masses
Suspicious descriptors: irregular shape, non-circumscribed margin, rim/heterogeneous enhancement, washout kinetics.
- 0 suspicious → BI-RADS 3
- 1 suspicious → BI-RADS 4A
- 2 suspicious → BI-RADS 4B
- 3 suspicious → BI-RADS 4C
- 4 suspicious → BI-RADS 5

## Scoring Logic for NME
Suspicious descriptors: segmental/diffuse distribution, clumped/clustered-ring pattern, washout kinetics.
- 0 suspicious → BI-RADS 3
- 1 suspicious → BI-RADS 4A
- 2 suspicious → BI-RADS 4B
- 3 suspicious → BI-RADS 4C/5
