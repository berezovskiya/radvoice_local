# Bosniak Classification v2019 — Reference

## Citation
Silverman SG, Pedrosa I, Ellis JH, et al. Bosniak Classification of Cystic Renal Masses, Version 2019: An Update Proposal and Needs Assessment. Radiology. 2019;292(2):475-488. DOI: 10.1148/radiol.2019182646. PMID: 31210616.

## Prerequisite
Cystic renal mass = < 25% enhancing tissue. >= 25% = solid mass, not Bosniak.
Enhancement: CT >= 20 HU increase; MRI >= 15% signal increase.

## Classification Criteria

### Bosniak I — Simple Cyst
- Thin (<=2mm) smooth wall, no septa
- Homogeneous simple fluid (-9 to 20 HU)
- Malignancy: 0%. Management: No follow-up.

### Bosniak II — Minimally Complex
- Thin (<=2mm) smooth wall
- 0-3 thin smooth septa (enhancing OK)
- Homogeneous non-enhancing >20 HU, OR >=70 HU non-contrast <=3cm
- Calcification of any type permitted
- Malignancy: <1%. Management: No follow-up.

### Bosniak IIF — Probably Benign
- >=4 thin (<=2mm) smooth enhancing septa, OR
- Smooth minimally thickened (3mm) enhancing wall/septa
- Malignancy: ~10%. Management: Surveillance 6mo, 12mo, then yearly x5yr.

### Bosniak III — Indeterminate
- Enhancing thick (>=4mm) smooth wall/septa, OR
- Enhancing irregular wall/septa (convex protrusion <=3mm, obtuse margins)
- Malignancy: ~50%. Management: Surgery or active surveillance.

### Bosniak IV — Likely Malignant
- Enhancing nodule(s): convex protrusion >=4mm (obtuse margins) OR any size (acute margins)
- Malignancy: ~85-90%. Management: Surgery recommended.

## Key Measurements
| Thickness | Class |
|-----------|-------|
| <=2mm | Thin (I, II) |
| 3mm | Minimally thickened (IIF) |
| >=4mm | Thickened (III) |

## Septa Count
| Count | Class |
|-------|-------|
| 0 | I |
| 1-3 thin smooth | II |
| >=4 thin smooth | IIF |

## Calcification
NOT an independent upgrading criterion in v2019.
